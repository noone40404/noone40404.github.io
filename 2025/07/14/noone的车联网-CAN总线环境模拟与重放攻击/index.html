  • 环境

    • ICSim 仪表盘模拟器
    • Socketcand can网络

    • Kayak 基于socketCAN的分析工具

      注意要用低版本的jdk，否则会导致缺少 Pack200 而编译错误

  • 启动模拟器

    在 ICSim 中

    此时可以用键盘或者手柄控制：

    | 功能 | 控制按钮 |
    |—————————|———————————|
    | 转向 | 键盘左右 |
    | 速度 | 键盘上下 |
    | 开/关左前车门 | 右Shift / 左Shift + A |
    | 开/关右前车门 | 右Shift / 左Shift + B |
    | 开/关左后车门 | 右Shift / 左Shift + X |
    | 开/关右后车门 | 右Shift / 左Shift + Y |
    | 开启全部车门 | 左Shift + 右Shift |
    | 关闭全部车门 | 右Shift + 左Shift |
