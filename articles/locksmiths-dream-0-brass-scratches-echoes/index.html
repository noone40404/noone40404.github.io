梦见黄铜，划痕与回声

发表于 2024-10-07 更新于 2026-07-07 分类于 锁匠的梦境 阅读次数：
每个锁匠迟早会做梦。现在我们来背诵它的七个阶段吧……

  • 老式一字锁

    最简单的锁，有种新手村史莱姆的美（雾

    扭力扳手（大概这么叫吧）加单钩，扭力扳手扭转的同时用单钩一遍遍刷直至刷开

    如果多次未刷开则放开扭力扳手让锁芯复位后重试

  • 一字锁2.0

    突然发现除了上面那种圆的还有个长条的双头练功锁，长得都一样，开法也一样

  • 一字锁3.0

    还有个超大号的锁，我还以为是哪个高级锁，但看着像一字锁就随手刷了刷，然后直接就开了啊嗯

  • 十字锁

    还是懒狗快开工具好用啊

    就直接插到底，工具上四个突起卡住锁的十字充当扳手，然后扭的同时反复抽插就开了

    唯一要注意的是因为要腾出来一个手拿着锁，所以不好发力

    正常开门锁应该很方便，但练功锁我是用腿夹着两个手一起拧才打开的（大嘘。两个手拿着确实没办法发力拧开

  • 无簧挂锁/叶片锁

    大概就是这种方形的锁孔，里面无弹簧

    用的工具还是懒狗快开：

    插到底，对扭矩施力旋转同时反复扭动。

    可惜我自己之前用的锁太小了快开伸不进去

  • 卡巴锁

    这个有难度的，开了好几天记录一下心得：

    由于没有快开工具，让我们回到开锁的本质：

    即通过单钩点每个锁点加上扭力扳手的旋转力使得锁芯卡在正确的位置，每个锁芯是由两个分开的小轴叠起来的，正确的钥匙可以让每个锁芯的两个轴之间的缝隙恰好在同一直线并可以转动

    先参考一下锁住的正常状态：

    对于锁芯，存在三种状态忽略我现编的名字，搭配图片食用：

    • 常规态（即拔下钥匙后的正常上锁状态）：上锁轴高，需要施加一定量的力才能使单钩穿过
    • 到位态：通过练功锁侧面可以勉强看见上锁轴，此时由于扭力扳手的力，锁芯被卡在正确的位置，只要扳手不松基本不会复位

    • 越位态：单钩转动时用力过猛，旋转 $90°$ 以上会出现该情况，此时上下锁轴均被卡在下面，侧面上方看不见上锁轴，而弹簧位置可以看到上锁轴挡住转动处。这种状态的锁芯并不稳定，敲击锁底侧即可令其归位

      而锁的到位则是有顺序的，并非从右到左，实际开锁时使用单钩慢慢点，反复点还回弹的话就越过去点下一个，一遍点完再倒回来点，直至点下去所有点，单钩在里面畅通无阻，即下图：

      可以看出来此时所有锁芯均已到位，看起来没到位那两个是角度问题，其实单钩已经基本不受阻力了

      但实战情况下看不到锁芯情况，判断标准就是在使劲扳住扭力扳手的时候敲击锁底部数次，来避免发送越位态情况，此时如果扔畅通无阻就是全部到位

      此时继续用单钩反复来回刷几下就开了

      这个多练几天就好了
